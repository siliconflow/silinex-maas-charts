# silinex-maas-charts

A Helm charts repository for Silinex MAAS project.

## Getting Started

```bash
# Add this repository
helm repo add silinex-maas-charts https://github.com/GitHub007Dra/silinex-maas-charts

# Update repos
helm repo update

# Search charts
helm search repo silinex-maas-charts
```
